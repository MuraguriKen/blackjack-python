## Blackjack-Python

A console-based Blackjack game built in Python using object-oriented programming.

## Features
- Full 52-card deck generation and shuffling
- Player vs Dealer gameplay
- Bank and betting system
- Ace value logic (1 or 11)
- Hit/Stay decision flow
- Win/Loss/Tie handling

## Skills Demonstrated
- Class design and object interaction
- Game state management
- Conditional logic
- Input validation
- Procedural game flow

## How to Run
Run the notebook or Python file to start the game.
